# RoadSceneï¼ša new dataset of aligned infrared and visible images

This datset has 221 aligned Vis and IR image pairs containing rich scenes such as roads, vehicles, pedestrians and so on. These images are highly representative scenes from the [FLIR video](https://www.flir.com/oem/adas/adas-dataset-form/) with highly representative scenes. We preprocess the background thermal noise in the original IR images, accurately aligned the Vis and IR image pairs, and cut out the exact registration regions to form this dataset.<br>
